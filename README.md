# gostdoc

[![CI](https://github.com/samikofficial/gostdoc/actions/workflows/ci.yml/badge.svg)](https://github.com/samikofficial/gostdoc/actions/workflows/ci.yml)

CLI и Python-библиотека: приводят оформление `.docx` (доклад, курсовая, ВКР, отчёт о НИР)
к **ГОСТ 7.32-2017**. Меняется только оформление — текст автора остаётся побайтно прежним.

## Установка (для разработки)

```
python -m venv .venv
.venv\Scripts\activate
pip install -e .[dev]
```

## Использование

```
gostdoc INPUT.docx [-o OUTPUT.docx] [--check] [--no-detect-structure]
```

> Без установки Python можно собрать автономный `gostdoc.exe` — см. [packaging/build.md](packaging/build.md).

- Без `-o` результат пишется рядом: `INPUT.gost.docx` (исходник не перезаписывается).
- `--check` ничего не пишет: проверяет соответствие ГОСТ-константам, печатает несоответствия,
  код выхода `0` (соответствует) или `1` (есть отклонения).
- Распознавание неразмеченной структуры (заголовки `N.N`, главы, структурные элементы,
  подписи) включено **по умолчанию** и логируется. Отключить — `--no-detect-structure`
  (тогда выполняется только чистое v0-форматирование оформления).

### Переопределения под методичку вуза

По умолчанию применяется базовый ГОСТ 7.32-2017. Если методичка вуза отличается,
параметры можно переопределить (расходятся обычно только поля, номер страницы и
полужирность заголовков):

```
gostdoc ДИПЛОМ.docx --margins 30,10,20,20        # поля Л,П,В,Н в мм
gostdoc ДИПЛОМ.docx --page-number bottom-right   # bottom-center|bottom-right|top-right|none
gostdoc ДИПЛОМ.docx --page-number-size 12        # кегль номера страницы
gostdoc ДИПЛОМ.docx --no-bold-headings           # заголовки без полужирного
```

`--check` учитывает те же переопределения.

## Что делает v0

Поля, шрифт, кегль, межстрочный интервал, абзацный отступ, выравнивание, нумерация страниц,
оформление существующих стилей заголовков и подписей — по ГОСТ 7.32-2017.

Чего v0 **не** делает: не распознаёт неразмеченные заголовки, не правит список литературы,
не учитывает методички вузов, не конвертирует `.doc` → `.docx`, не трогает орфографию.

См. `ТЗ.md` и `CLAUDE.md`.
